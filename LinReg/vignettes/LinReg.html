Intro to LinReg

## Warning: package 'ggplot2' was built under R version 3.4.4

When working with linear regression models you must:

  • Let the computer do the calculations (much faster!)

  • Describe the formula for the model and specify data

  • Execute the program

The LinReg package makes these steps fast and easy. This document introduces you to LinRegâ€™s basic set of tools, and shows you how to apply them to your dataset.

Example

linreg(formula = Sepal.Width ~ Sepal.Length, data = iris)
